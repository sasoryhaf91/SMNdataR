# National 1-km grid for Mexico with altitude

1-km lattice nodes covering Mexico with altitude.

## Usage

``` r
gridmex
```

## Format

A data frame with columns:

- station:

  character: synthetic node id

- latitude:

  numeric, WGS84

- longitude:

  numeric, WGS84

- altitude:

  numeric, meters
